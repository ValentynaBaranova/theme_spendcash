Шрифт Inter — який файл додати
================================

У розпакованому архіві Inter-4.1 (як на вашому скріні):

  Додайте в цю папку ОДИН файл з кореня архіву:

    InterVariable.ttf

  Тобто: скопіюйте InterVariable.ttf з папки Inter-4.1 сюди:
  wp-content/themes/spendvest/assets/fonts/

Інші файли з архіву не потрібні:
  • Inter.ttc — для десктопу
  • Папка web/ — окремі навантаження (Regular, Bold тощо); можна не використовувати
  • InterVariable-Italic.ttf — курсив, додайте лише якщо потрібен окремо

Після копіювання InterVariable.ttf шрифт Inter на сайті працюватиме.
